# aenir

## A Fire Emblem statistics calculator
This module lets you do stat calculations for Fire Emblem units.
With this module, you can:
- Create a virtual copy of a unit and their stats.
- Level up that unit.
- Promote that unit.
- Compare one unit to another.
- And much more!

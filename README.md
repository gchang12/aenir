HOW TO USE
==========
```python
# create Morph object
from aenir.morph import Morph6
roy = Morph6("Roy")
# specify target level
roy.level_up(20)
# cap stats wrt unit's maximum-stats for their class
roy.cap_stats()
```

```python
# create Morph objects to compare stats
from aenir.morph import Morph6
allen = Morph6("Allen")
```
